🚀 Swiggy Clone – Food Ordering UI

🔗 Live Demo: https://i-clone-swiggy.netlify.app/

📂 GitHub Repo: https://github.com/Yatender-zoro/Swiggy_Clone

A fully responsive Swiggy UI clone built using modern frontend architecture.
This project integrates live Swiggy APIs, implements global state management using Redux Toolkit, and demonstrates scalable React application structure.

⚛ React (Functional Components + Hooks)

🎨 Tailwind CSS

📦 Parcel (Bundler)

🗂 React Router DOM (Dynamic Routing)

🧠 Redux Toolkit (Global State Management)

🌐 Live Swiggy APIs Integration

🔐 CORS Handling

📱 Fully Responsive Design

⚙️ Installation & Setup

Clone the repository:

git clone https://github.com/Yatender-zoro/Swiggy_Clone.git
cd Swiggy_Clone

Install dependencies:

npm install

Run locally:

npm start

Build for production:

npm run build
